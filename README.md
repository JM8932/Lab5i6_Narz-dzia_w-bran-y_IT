# Konwerter Danych (JSON/YAML/XML)

Autor: Jakub Merkel 60352

## Opis projektu
Program konsolowy służący do konwersji danych między formatami:
- JSON (.json)
- YAML (.yml, .yaml)
- XML (.xml)

Program automatycznie rozpoznaje formaty plików wejściowych i wyjściowych na podstawie ich rozszerzeń.

## Sposób użycia
Aby uruchomić program, użyj terminala (wiersza poleceń):
